Learn to create beautiful websites

Introduction

This website is a resource for learning the technique and technology used to make beautiful websites. Learn fundamental design techniques such as color theory, grids, composition and typography, and learn to apply them to the web using HTML, CSS, and JavaScript.

Note: This website is currently being developed and not all the content is completed. If you are interested in seeing the work so far, visit the full list of articles.

Who is this for?

Resources, tutorials, and references are available for web designers and web developers with a broad range of skill sets. This website assumes nothing but a desire to create beautiful things on the web.

What will I learn?

The introduction to the web provides an overview and introduction to designing for the web. Go from beginner to master of HTML, CSS, and JavaScript in the web technologies track. Focus on learning design principles and how they apply to the web in the web design track. Finally, learn the process and practice of creating and designing websites professionally with the web professional track.

Sounds awesome! Where do I start?

If you are new to web design and development, start at the beginning: an introduction to designing for the web. This will lay the groundwork and get you started on the right path.

If you want a better understanding of how websites work and how to create them yourself, start at the web technologies track. Review the outline for each section and jump in at whatever starting point feels appropriate for your background and skill level

If you have a strong understanding of web technologies and a desire to create something beautiful, head over to the web design track to learn fundamentals of design, and how to apply those fundamentals to the web.

If you are interested in creating websites for a living, head to the web web professional track to learn the process and practice of creating web sites.

Excellent! Wait, who in the world are you?
