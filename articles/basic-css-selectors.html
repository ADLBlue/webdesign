What are CSS Selectors?

CSS enables a user to style their HTML code to add more creativity and personality to a web page. There are many types of CSS selectors but we will just focus on a few basics to start with. It assigns specific attributes to different elements within the web code. Below is how selectors will look in CSS.


selector {
  property1 : value;
}

Lets Create

Creating headlines are a good place to start to learn how CSS code affects how your web page will look. Below you can see the tag <h1> in HTML, this will be the selector code used in CSS to adjust how the web page headline will look.

Now going to CSS, start with “h1” and an open curly bracket to begin assigning attributes. Font-family, color, and font-size are just a few things that can be adjusted. Don’t forget to also close with a curly bracket.
