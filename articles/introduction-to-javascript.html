A few things you have to know before you jump in JavaScript.

There are many ways to make dynamic websites. JavaScript is one of them, and most websites use JavaScript. Recently web developers can use JavaScript as server-side script.

What is JavaScript?

JavaScript is the prototype-based scripting language of the web to improve user-interfaces and build websites with dynamic features. For example, auto-fill forms, auto refresh, and so on. JavaScript is not same as Java even though both are based on C and C++. Java developed by Sun is a programming language that does totally different things, and Java is much harder to learn. JavaScript is developed by Brendan Eich and working at Netscape.

Why JavaScript?

JavaScript is lightweight and has excellent portability. Also it interacts with the sites and works well with HTML and CSS.
